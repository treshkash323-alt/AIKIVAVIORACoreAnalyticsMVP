# AIKIVAVIORA Core Analytics MVP

Учебно-практический проект аналитического ядра экосистемы AIKIVAVIORA.

## Назначение

Проект создаётся в рамках домашнего задания по Supabase, Google AI Studio и BI-аналитике.

Цель проекта:
- изучение Supabase;
- работа с PostgreSQL;
- создание RPC-функций;
- построение BI-dashboard;
- интеграция AI assistant;
- подготовка reusable CORE-компонентов для экосистемы AIKIVAVIORA.

## Архитектура

- Supabase
- PostgreSQL
- RPC Functions
- Google AI Studio
- Gemini
- Dashboard Analytics
- AI Assistant
- SQL Layer

## Статус

MVP / Learning Stage
