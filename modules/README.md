# Modules

Здесь живут прикладные направления экосистемы AIKIVAVIORA.

- `analytics_agent` — отдельная логика аналитического агента
- `rag` — пайплайн retrieval augmented generation
- `telegram_bots` — Telegram-боты и сценарии общения
- `data_connectors` — подключения к Supabase и другим источникам данных

Правило простое: если это не общий core-код и не одноразовый эксперимент, скорее всего это модуль.
